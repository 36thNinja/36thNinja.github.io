Reset IE Web Settings

  1. In order to reset IE web settings we need to close all Internet explorer browsers.

  2. Click on Start -> Type 'Internet' in Search Programs and files and then select 'Internet Options'

  3. Select the tab 'Advanced' and then click 'Reset'

  4. On the pop up window please check 'Delete Personal settings' and then click 'Reset'

  5. Please click select Close.

  6. Now try accessing Direct office.
